Cytoscape 3 Developer Tutorial Cheat Sheet

Maven

  • General usage 
    mvn «goal-1» [ «goal-2» ... ]
  • Clean your project directory 
    mvn clean
  • Build your project 
    mvn install
  • Clean, then build your project 
    mvn clean install

  • Note: If you are working offline and need to compile/run the starter code, you'll need to copy the provided MavenRepo directory to your home directory.

    Mac/Linux

    mkdir ~/.m2
cp -r StarterCode/MavenRepo ~/.m2/repository

    Windows

    1. Create a folder called .m2 in your home folder (i.e. the one that contains your Desktop folder.
    2. Copy the folder StarterCode\MavenRepo to .m2.
    3. Rename the folder .m2\MavenRepo to .m2\repository.

Cytoscape

  • API documentation
  • Launch Cytoscape

    Mac/Linux

    ./cytoscape.sh

    Windows

    cytoscape.bat
  • Launch Cytoscape with remote debugging enabled

    Mac/Linux

    ./cytoscape.sh debug

    Windows

    cytoscape.bat debug

Cytoscape OSGi Shell

  • Show a list of all available commands
    • help
  • Show documentation for a specific command
    • help «command-name»
  • List all bundles
    • list
  • List all services
    • ls
  • List services registered by a bundle
    • ls «bundle-id»
  • Install a bundle from Maven 
    install mvn:«group-id»/«artifact-id»
    e.g.: 
    install mvn:com.example/HelloWorld
  • Start a bundle 
    start «bundle-id»
    e.g.: 
    start 169
  • Reload (and restart) a running bundle 
    update «bundle-id»
    e.g.: 
    update 169

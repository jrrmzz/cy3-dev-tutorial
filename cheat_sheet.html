Cytoscape 3 Developer Tutorial Cheat Sheet

Maven

If you have a working internet connection:

  • General usage 
    mvn «goal-1» [ «goal-2» ... ]
  • Clean your project directory 
    mvn clean
  • Build your project 
    mvn install
  • Clean, then build your project 
    mvn clean install

If you are working offline:

  • General usage 
    mvn -o «goal-1» [ «goal-2» ... ]
  • Using a custom local repository (default is ~/.m2 on Mac/Linux, or %USERPROFILE%\.m2 on Windows)

    Mac/Linux

    mvn -o -Dmaven.repo.local=/path/to/repository «goal-1» [ «goal-2» ... ]

    Windows

    mvn -o -Dmaven.repo.local=path\to\repository «goal-1» [ «goal-2» ... ]
  • Build your project 
    mvn install
  • Clean, then build your project 
    mvn clean install

Cytoscape

  • Launch Cytoscape

    Mac/Linux

    ./cytoscape.sh

    Windows

    cytoscape.bat
  • Launch Cytoscape with remote debugging enabled

    Mac/Linux

    ./cytoscape.sh debug

    Windows

    cytoscape.bat debug

Cytoscape OSGi Shell

  • Install a bundle from Maven 
    install mvn:«group-id»/«artifact-id»
    e.g.: 
    install mvn:com.example/HelloWorld
  • Start a bundle 
    start «bundle-id»
    e.g.: 
    start 169
  • Reload (and restart) a running bundle 
    update «bundle-id»
    e.g.: 
    update 169
